# LessTest: The Future of Frictionless Testing

LessTest is built on a single philosophy: **Testing should be the easiest part of your workflow.** While legacy tools add weight and complexity, LessTest stays out of your way, delivering speed and clarity without the boilerplate.

## üöÄ Key Features

- **Zero Config:** Automatically detects your project structure (Next.js, Vite, Node) and configures the environment instantly.
- **Extreme Parallelization:** Built from the ground up to utilize every core of your CPU. Tests that take minutes in Playwright take seconds in LessTest.
- **Human-Centric Logs:** No more "stack trace soup." LessTest provides clean, visual diffs and clear explanations of failure points.
- **Intelligent Watch Mode:** A lightning-fast reactive loop that only runs the tests affected by your current code changes.
- **Native Type Safety:** Deep TypeScript integration providing true auto-completion for selectors and actions.
- **Visual Snapshotting:** Modern visual regression testing baked directly into the core runner.
